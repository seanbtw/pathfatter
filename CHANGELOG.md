# Changelog

All notable changes to PathFatter will be documented in this file.

## [Unreleased]

## [0.3.0] - 2026-04-11

### Added
- Phase 3: Advanced features & accessibility
  - VoiceOver support with custom labels
  - Dynamic Type scaling
  - High contrast mode adaptation
  - Reduced motion support
  - Keyboard-only navigation

## [0.2.0] - 2026-04-11

### Added
- Phase 2: UX improvements
  - History grouped by date (Today, Yesterday, This Week, Older)
  - Pin frequently-used paths
  - Search through history
  - Keyboard navigation for history

## [0.1.0] - 2026-04-11

### Added
- Phase 1: Visual polish overhaul
  - Glassmorphism design with animated gradient orbs
  - Time-based accent colors
  - Spring animations
  - Syntax highlighting for path components

### Changed
- Code quality improvements from initial release

## [0.0.1] - 2026-04-11

### Added
- Initial release: PathFatter macOS app
- Windows ↔ macOS path conversion
- SharePoint URL → local OneDrive path conversion
- UNC/SMB path support
- Custom drive mappings
- Safari web extension for SharePoint URL detection
- 4-step onboarding carousel
