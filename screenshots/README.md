## 📤 How to Upload

1. Go to [App Store Connect](https://appstoreconnect.apple.com)
2. Select your PathFatter app
3. Click "App Store" tab
4. Scroll to "Screenshots" section
5. Select display size (13-inch first)
6. Drag and drop screenshots
7. Repeat for 15-inch display
8. Add optional captions if desired
9. Save
